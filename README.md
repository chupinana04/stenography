# Labtainer Scene Change Analysis

This repository hosts an imodule tar for the `scene-change-analysis` Labtainer lab.

Install in a Labtainers VM:

```bash
imodule https://github.com/chupinana04/stenography/raw/refs/heads/main/video/scene-change-analysis.tar
labtainer -r scene-change-analysis
```

Docker image used by the lab:

```text
dattitbx/scene-change-analysis.scene-change-analysis.student:latest
```
